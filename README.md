# mao-decide

**相信你也感到迷茫，运用《毛选》的方法论为你分析以便你自己做决定。**

---

## 这是什么

mao-decide 是一个 AI 决策分析工具，作为 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) 的 Skill 运行。

它不是算命，不是心灵鸡汤，也不是那种"两边都有道理，你自己看着办"的废话生成器。

它做的事情很简单：**帮你把问题想清楚，然后给你一个明确的判断。**

方法论来自毛泽东选集——不是掉书袋式的引经据典，而是把毛泽东处理复杂问题的思维方式，应用到你的具体决策中。同时会告诉你分析思路的出处，引导你去读原文，形成自己的判断力。

## 为什么做这个

市面上的决策工具和 AI 助手有一个通病：**不敢下判断。**

你问"我该不该辞职"，它告诉你"这取决于你的具体情况"。你问"A还是B"，它告诉你"两边都有道理，建议综合考虑"。听起来很对，但毫无价值——你要是能综合考虑，还来问它干嘛？

更糟糕的是那种看似给了建议实则在回避的回答："不必二选一，可以先小步验证。"表面上找到了选项C，实际上是在回避主要矛盾，给了一个不痛不痒的答案。

毛泽东做决策的特点恰恰相反：**敢判断，敢押注，敢在信息不完整的时候做决定。** 但这种"敢"不是莽撞——它建立在深入的调查研究和严密的矛盾分析之上。

这就是 mao-decide 要做的事。

## 核心流程

### 第一步：调查研究（搜集事实）

> "没有调查就没有发言权" —— 《反对本本主义》

你描述决策后，AI 进入"调查员"模式，逐个提问搜集信息：

- 客观事实是什么？（数字、时间线、已发生的事）
- 涉及哪些人？各方的立场和利益？
- 有哪些硬约束？（时间、资源、不可逆条件）
- 之前尝试过什么？结果如何？
- 哪些信息你不确定但可能重要？

关键动作：**帮你区分"事实"和"感受"。** 你说"他们不会同意的"，它会追问："这是你的判断还是他们明确表达过的？"

### 第二步：矛盾分析（看清本质）

> "研究任何过程，如果是存在着两个以上矛盾的复杂过程的话，就要用全力找出它的主要矛盾。" —— 《矛盾论》

基于搜集的事实，识别决策中的所有矛盾，然后判断：

- **主要矛盾是什么？** 解决了它，其他问题会随之缓解
- **矛盾的主要方面是什么？** 它决定了行动方向
- **你最初的问题框架对不对？** 很多人纠结于表面选择，真正的矛盾藏在更深处

这一步是核心价值所在。但如果你的判断本身就是准确的，AI 会直接确认，不会为了显得深刻而强行翻转你的认知。

### 第三步：实践策略（行动方案）

> "实践、认识、再实践、再认识，这种形式，循环往复以至无穷" —— 《实践论》

给出明确的判断和可执行的行动建议：

- **明确方向**：该选A就说A，该选B就说B，不和稀泥
- **具体行动**：第一步做什么，第二步做什么
- **可团结的力量**：谁能帮你？哪些资源可以调动？
- **风险标注**：需要持续观察什么
- **推荐阅读**：与你的决策最相关的毛选文章，帮你建立自己的分析能力

## 四大原则

| 原则 | 说明 |
|------|------|
| 不说正确的废话 | 敢于下判断。"两边都有道理"是禁止输出。 |
| 认知纠偏是发现不是预设 | 你看得准就确认，有盲区才纠偏。 |
| 活用智慧，标注出处 | 内化思维方式的同时告诉你出处，引导你读原文。 |
| 有温度但不和稀泥 | 可以共情你的处境，但不会因此回避尖锐判断。 |

## 覆盖的决策场景

| 场景 | 核心思维 |
|------|----------|
| 两难选择 | 表面的A vs B背后，往往藏着真正的矛盾C |
| 面对强敌/不利局面 | 战略上藐视，战术上重视；集中优势兵力 |
| 信息不足时决策 | 没有调查就没有发言权，但调查也有优先级 |
| 多方利益冲突 | 谁是朋友谁是敌人，这是首要问题 |
| 要不要改变现状 | 量变积累到质变，关键是判断临界点 |
| 失败/挫折后复盘 | 批评与自我批评不是自我否定，是找到下一步 |
| 长期vs短期取舍 | 持久战思维：战略耐心 + 战术果断 |

## 安装与使用

### 前提

你需要安装 [Claude Code](https://docs.anthropic.com/en/docs/claude-code)（Anthropic 官方 CLI / IDE 扩展）。

### 安装

将 `skill/` 目录复制到你的 Claude Code skills 目录：

```bash
# macOS / Linux
cp -r skill/ ~/.claude/skills/mao-decide/

# Windows
xcopy skill\ %USERPROFILE%\.claude\skills\mao-decide\ /E /I
```

或者手动将 `skill/` 文件夹下的三个文件复制到 `~/.claude/skills/mao-decide/` 目录中。

### 使用

在 Claude Code 中直接调用：

```
/mao-decide 我在考虑要不要辞职创业
```

```
/mao-decide 公司给了管理岗机会但要降薪，家人反对
```

```
/mao-decide 两个offer怎么选
```

也可以用中文触发词：

```
/矛盾决策 我该不该转行
```

AI 会根据你的决策复杂度自动调整流程——简单问题快速分析，复杂问题走完整三步。

## 文件结构

```
skill/
├── SKILL.md           # 主 skill 定义（流程指令、四大原则、输出格式）
├── methodology.md     # 方法论知识库（7个决策场景 × 思维模式 × 历史案例 × 来源文章）
└── examples.md        # 完整分析示例（含认知纠偏型和确认判断型两种）
```

## 不是什么

- **不是算命工具** — 它帮你分析，但最终决定是你自己做的
- **不是心灵鸡汤** — 它会给你不想听的判断
- **不是毛选读书笔记** — 方法论是活的，不是教条
- **不是万能的** — 如果信息不够判断，它会诚实告诉你还需要搞清楚什么
