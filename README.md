## 为什么

AI 助手的通病：不敢下判断，给出模棱两可的"正确的废话"——

> "这取决于你的具体情况，两边都有道理，建议你综合考虑。"

毛泽东做决策的特点是敢判断、敢押注。但不是莽撞——建立在调查研究和矛盾分析之上。

这个 skill 把毛泽东的方法论内化为分析的操作系统，不是贴标签：

| 分析阶段 | 驱动文章 | 核心操作 |
|---|---|---|
| 调查研究 | 《实践论》+《反对本本主义》 | 区分感性认识（直接经历）和理性认识（推断），没有调查就没有发言权 |
| 矛盾识别 | 《矛盾论》 | 从事实中归纳张力，识别伪张力，找根因矛盾 |
| 矛盾分析 | 《矛盾论》 | 主要矛盾、矛盾的主要方面、同一性（转化条件）、特殊性 |
| 实践策略 | 《论持久战》+《中国社会各阶级的分析》 | 力量对比四象限、阶段判断（防御/相持/反攻）、利益相关方分类 |

## 方法论场景库

这是这个 skill 区别于普通决策框架的核心：**7 个从《毛选》提炼的决策场景**，每个场景都有具体的历史案例作为推理依据，并明确标注"可借鉴的推理逻辑"和"不可直接套用的部分"。

| # | 场景 | 核心方法论 | 历史案例 |
|---|---|---|---|
| 1 | **面对两难选择** | 找到 A vs B 背后的深层矛盾 C，打破框架 | 抗战初期"速胜论 vs 亡国论"→持久战 |
| 2 | **面对强敌、不利局面或被动处境** | 整体劣势不等于局部无法创造优势 | 井冈山十六字诀、重庆谈判 |
| 3 | **多方利益冲突与内部矛盾** | 先判断敌我矛盾还是人民内部矛盾，方法完全不同 | 抗日统一战线、延安整风 |
| 4 | **要不要改变现状（含时机判断）** | 量变积累到质变临界点才是行动时机 | 秋收起义后转向农村 |
| 5 | **失败或挫折后的复盘** | 找决定性的主观原因，不是列一堆原因 | 遵义会议 |
| 6 | **资源极度有限时的分配** | 集中优势兵力，找最薄弱的突破口，不是最重要的 | 三大战役先打锦州 |
| 7 | **不确定性极高时的决策** | 先做最小代价的验证，不是等信息全了再行动 | 《反对本本主义》调查方法论 |

每个场景的完整内容见 [skill/methodology.md](skill/methodology.md)。

## 三步流程

```
调查研究 → 矛盾识别 → 矛盾分析 → 实践策略
```

**Step 1 调查研究**

逐一追问，区分你亲身经历的事实（感性认识）和你的推断（理性认识）。当你说"他们不支持我"，会追问"他们具体说了什么或做了什么"。调查结束输出摘要，让你确认后再进入分析。

**Step 1.5 矛盾识别（内部步骤）**

从调查事实中归纳所有张力，识别伪张力（外部压力被误认为内部矛盾），分析张力之间的因果关系，筛选出主要矛盾。这一步是 Step 1 和 Step 2 之间的关键桥接。

**Step 2 矛盾分析**

用《矛盾论》的概念真正分析，不是贴标签：
- 主要矛盾是什么，为什么解决它其他问题会随之缓解
- 矛盾的主要方面——决定走向的那一面（不是"更强的那一面"）
- 同一性分析——当前的劣势在什么条件下会转化为优势
- 如果你的问题框架有偏差，指出偏差在哪里、为什么

**Step 3 实践策略**

- 力量对比四象限：我强在哪里、我弱在哪里、对手强在哪里、对手弱在哪里
- 阶段判断：战略防御（保存力量）/ 战略相持（积累优势）/ 战略反攻（主动出击）
- 利益相关方：核心盟友、可争取的中间力量、对立方/阻力
- 具体行动：按阶段排序，找最薄弱的突破口，集中资源

## 决策类型

skill 会在启动时识别你的决策类型，不同类型走不同路径：

**利弊决策**（需要分析选项优劣）→ 走标准三步流程

**元决策**（你已经在内心做了选择，需要确认自己是否真的想清楚了）→ 不分析利弊，检验你的决定是否基于充分的感性认识，以及你是否真正理解了后果

**价值观决策**（没有外部对手，核心是个人价值取舍）→ 不用力量对比框架，帮你检验意愿的真实性和对后果的理解

## 四大原则

**原则一：不说正确的废话，敢于下判断**

禁止输出"两边都有道理"、"这取决于你的具体情况"、"建议你综合考虑"。判断必须从矛盾分析的结论中直接推导。

置信度处理：
- 高置信度 → 直接给出明确判断
- 中置信度 → 给出判断，标注"如果 X 信息改变，结论会变"
- 低置信度（感性认识严重缺失）→ 不下最终判断，但给出行动判断："你现在不应该做这个决定，而应该先做[具体的最小验证]"

**原则二：认知纠偏是发现，不是预设**

用户看得准 → 直接确认，快速进入策略。用户有盲区 → 才纠偏，说清楚偏差在哪里、为什么。不为了显得深刻而强行翻转认知。

**原则三：方法论驱动分析，不是事后标注**

毛的方法论在分析过程中真正发挥作用，不是分析完了再贴标签。

**原则四：有温度但不和稀泥**

可以共情，但不能因为共情就回避尖锐的判断。

║         决策：要不要辞职创业             ║

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

★ 主要矛盾
  你对 AI 教育方向的判断（理性认识）
  与
  你从未直接验证过这个判断（感性认识严重缺失）
  之间的对立

  └ 主要方面：感性认识的缺失是决定性的。
    热情是真实的，但热情不等于市场机会。
  └ 同一性分析："没有验证"这个劣势，在什么条件下会转化？
    ——当你用 2-3 个月的业余时间完成最小验证后。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 明确判断：现在不应该辞职。

  不是因为创业不好，而是因为你还没有完成
  最基本的实践检验。在没有感性认识支撑的情况下
  做最大的决策，不是勇气，是赌博。
