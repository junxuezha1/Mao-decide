# mao-decide 方法论知识库

本文件按决策场景组织毛泽东的思维模式。分析时自然融入，不要机械套用。每个场景标注了核心来源文章，用于引导用户阅读。

---

## 场景一：面对两难选择

**核心思维：表面的A vs B背后，往往藏着真正的矛盾C**

毛泽东很少在别人设定的选项里做选择。他的习惯是：先搞清楚为什么会出现这两个选项，找到背后的真正矛盾，然后从矛盾本身出发制定策略。

**分析路径：**
1. 用户说"A还是B"时，先问：为什么是这两个选项？是谁设定的？
2. 找到A和B背后共同指向的深层矛盾
3. 判断这个矛盾的主要方面——它决定了方向
4. 可能的结果：选A、选B、或者发现问题根本不是A vs B

**历史案例：**
- 1927年大革命失败后，党内争论"继续合作还是彻底决裂"，毛泽东看到的是"城市还是农村"这个更根本的问题——不在别人的框架里选择
- 抗战初期"速胜论vs亡国论"，毛泽东提出持久战——拒绝在两个错误选项中选一个

**核心来源：**
- 《矛盾论》——主要矛盾与次要矛盾的关系
- 《论持久战》——如何跳出错误的二元框架
- 《中国革命战争的战略问题》——在具体条件下判断方向

## 场景二：面对强敌或不利局面

**核心思维：战略上藐视，战术上重视；找到杠杆点，集中优势兵力**

不利局面下最容易犯两个错误：要么被吓住不敢动，要么急躁冒进想一把翻盘。毛泽东的方法是：承认整体劣势，但在局部创造优势。

**分析路径：**
1. 客观评估力量对比——不回避劣势，也不夸大劣势
2. 找到对手的弱点或局面中的杠杆点
3. 集中所有资源在杠杆点上发力
4. 战略上保持信心（大方向是对的），战术上极度谨慎（每一步都要算清楚）

**历史案例：**
- 井冈山时期：红军总兵力远不如国民党，但通过"敌进我退、敌驻我扰、敌疲我打、敌退我追"的十六字诀，在局部战斗中始终保持优势兵力
- 三大战役：全国层面兵力接近，但每一场具体战役都是集中优势兵力打歼灭战

**核心来源：**
- 《论持久战》——战略防御中的战术进攻思维
- 《集中优势兵力，各个歼灭敌人》——资源有限时的分配策略
- 《星星之火，可以燎原》——在极端不利时如何看到希望和路径

## 场景三：信息不足时如何决策

**核心思维：没有调查就没有发言权，但调查也有优先级**

毛泽东极度重视调查研究，但他不是说"等信息全了再决定"——很多时候信息永远不会全。关键是知道哪些信息是决定性的，优先搞清楚那些。

**分析路径：**
1. 区分"决定性信息"和"锦上添花的信息"
2. 决定性信息缺失 → 暂缓决策，先去调查
3. 非决定性信息缺失 → 可以在不确定中做判断，但标注风险
4. 调查的方法：不要只听一面之词，要多方交叉验证

**历史案例：**
- 1930年《反对本本主义》：批评那些不做调查就瞎指挥的人，但同时也批评那些"调查完了还不敢判断"的人
- 寻乌调查：毛泽东花大量时间做极其细致的社会调查，但调查的目的是为了做判断，不是为了拖延

**核心来源：**
- 《反对本本主义》——调查研究的方法论
- 《〈农村调查〉的序言和跋》——如何做有效的调查
- 《改造我们的学习》——理论联系实际，反对教条主义

## 场景四：多方利益冲突

**核心思维：谁是我们的朋友，谁是我们的敌人，这是革命的首要问题**

面对复杂的多方博弈，第一步不是"平衡各方利益"，而是搞清楚：在这个具体问题上，谁和你的核心利益一致，谁是对立的，谁是可以争取的中间力量。

**分析路径：**
1. 列出所有相关方及其核心利益
2. 按与用户利益的关系分类：盟友、对手、可争取的中间力量
3. 制定策略：团结盟友，争取中间，孤立对手
4. 注意：盟友关系不是永恒的，要根据具体问题动态判断

**历史案例：**
- 《中国社会各阶级的分析》：开篇就是"谁是我们的敌人？谁是我们的朋友？"——在复杂局面中先做最基本的分类
- 抗日统一战线：和昨天的敌人（国民党）合作，因为主要矛盾变了
- 解放战争时期：团结民主党派和中间力量，孤立蒋介石集团

**核心来源：**
- 《中国社会各阶级的分析》——利益相关方分析的经典框架
- 《论联合政府》——如何在复杂局面中建立最广泛的联盟
- 《目前形势和我们的任务》——动态调整策略

## 场景五：要不要改变现状

**核心思维：量变积累到质变，关键是判断临界点**

改变现状的决策最难的不是"要不要变"，而是"什么时候变"。太早变，条件不成熟会失败；太晚变，机会窗口关闭。

**分析路径：**
1. 当前的不满/问题是在积累还是在缓解？（量变方向）
2. 有没有到达临界点的迹象？（质变信号）
3. 外部条件是否支持改变？（时机判断）
4. 如果还没到临界点，该做什么准备？

**历史案例：**
- 秋收起义后转向农村：不是一开始就决定走农村路线，而是在城市起义失败后，根据实际情况做出的判断——条件变了，策略就要变
- "枪杆子里面出政权"：这个判断不是凭空产生的，是大革命失败这个"质变"事件后的认识飞跃

**核心来源：**
- 《矛盾论》——量变与质变的辩证关系
- 《实践论》——认识从实践中来，实践检验认识
- 《湖南农民运动考察报告》——如何判断变革的时机和力量

## 场景六：失败或挫折后的复盘

**核心思维：批评与自我批评不是自我否定，是找到下一步的方法**

失败后最常见的两种反应：要么全盘否定自己，要么找外部借口。毛泽东的方法是：承认错误，但重点是分析错误的原因，找到可以改进的具体环节。

**分析路径：**
1. 客观描述失败的事实（不夸大也不缩小）
2. 区分主观原因和客观原因
3. 主观原因中，哪些是可以改进的？（这才是复盘的价值）
4. 客观原因中，哪些下次可以提前预判？
5. 制定具体的改进措施，不是空泛的"下次注意"

**历史案例：**
- 第五次反围剿失败 → 遵义会议：不是简单地换人，而是深入分析了军事路线错误的具体原因
- 大跃进后的调整：承认错误，但重点是找到具体哪些政策需要调整

**核心来源：**
- 《论联合政府》中关于批评与自我批评的论述
- 《关于纠正党内的错误思想》——如何建设性地面对错误
- 遵义会议的历史经验——失败后如何做有效的路线调整

## 场景七：长期与短期的取舍

**核心思维：持久战思维——战略上的耐心 + 战术上的果断**

长期和短期的矛盾本质上是"当下的代价"和"未来的收益"之间的权衡。关键不是简单地"选长期"，而是判断：短期的代价你能不能承受？长期的收益是否确定？

**分析路径：**
1. 短期代价的具体内容和承受能力评估
2. 长期收益的确定性评估（是大概率还是小概率？）
3. 有没有办法降低短期代价同时保持长期方向？
4. 最坏情况下的退路是什么？

**历史案例：**
- 长征：短期看是巨大的损失和痛苦，但保存了革命的核心力量，为长期胜利奠定基础
- 抗日战争的持久战策略：明确告诉大家短期会很苦，但长期一定会赢，关键是怎么熬过去

**核心来源：**
- 《论持久战》——如何在长期困难中保持战略定力
- 《星星之火，可以燎原》——如何在短期劣势中看到长期趋势
- 《关于正确处理人民内部矛盾的问题》——如何平衡不同时间维度的需求
